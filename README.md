# learning-django

 ## Create venv
 ```
 python -m venv venv
 ```

 ## Init venv
 ```
 source venv/bin/activate
 ```
 ## Install django
 ```
 pip install django
 ```
 ## Create django project
 ```
 django-admin startproject {name_your_project}
 ```
 ## Make migrates
 ```
 python manage.py makemigrations
 ```
 ## Update migrates
 ```
 python manage.py migrate
 ```
 ## Create superUser
 ```
 python manage.py createsuperuser
 ```
